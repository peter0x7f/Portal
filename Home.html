# Requirement Success Criteria
1 The solution shall connect to an SQLite Database. Each forum will have its URL, icon and optionally username and password stored in the database in a retrievable manner.
2 The solution shall incorporate a web scraper. The Icon of the given URL will be downloaded.
3 The solution shall allow the user to navigate between saved websites Once a URL is given the website is saved and can be navigated to in the future
4 The solution shall allow users to add and remove saved websites The user can press a button and input, at minimum, the URL of the website
5 The solution should be written in Python It is written in Python
